# Lab 1 App

## Overview
This repo provides GPT integration with file I/O operations for processing text data.

## Features
- **GPT Integration** (`gpt.py`) - Interface with GPT models
- **File I/O** (`file_io.py`) - Read and write file operations
- **Main Entry Point** (`main.py`) - Core application logic

## Installation
1. Clone or download the repository
2. Navigate to the `src` directory
3. Install dependencies (if required)

## Usage
Run the application using:
```bash
make run
```

Or directly with:
```bash
python main.py
```

## Project Structure
```
src/
├── __init__.py
├── main.py          # Application entry point
├── gpt.py           # GPT model integration
├── file_io.py       # File read/write operations
└── Makefile         # Build automation
```

See `Makefile` for build and dependency information.
